Simulateur de trajet en java

Application web permettant de calculer le cout d'un trajet en fonction de la distance et la vitesse du véhicule réaliser en 2 versions

Informations sur le projet

  • Catégorie: Application JAVA
  • Client: Projet de formation
  • Date: Janvier 2025
  • Site web: GitHub

Objectifs:

Ce projet a pour objectif de créer une application web permettant de calculer le coût d'un trajet en fonction de la distance et la vitesse du véhicule dans une prémiere version puis d'ajouter des fonctionnalités pour une deuxième version (enregistrer un annonceur pour ajouter des publicités et afficher une synthèse ).
